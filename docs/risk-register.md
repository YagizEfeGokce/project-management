# Risk Register

| Risk | Type | Severity | Mitigation | Monitor |
| --- | --- | --- | --- | --- |
| Schedule overrun | Project | High | Sprint planning, scope control, buffer tasks | Burn-down, milestone slip |
| Technical debt growth | Technical | Medium | Code review, refactoring budget | Code quality metrics |
| Backend latency | Technical | Medium | Caching, query optimization, API profiling | Response time dashboard |
| Adoption resistance | Business | Medium | Simple UI, stakeholder feedback loop | Usage analytics |
| Data loss / sync issue | Technical | High | Backup state, validation, audit logs | Sync errors, backups |
