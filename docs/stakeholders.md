# Stakeholder Management

## Internal

- Project Manager: delivery visibility, sprint status, risk alerts
- Team Members: workload balance, task clarity, deadlines
- Tech Lead: architecture quality, technical debt, implementation risks

## External

- Client / Customer: business value, progress transparency, release confidence
- Sponsor / Management: ROI, schedule adherence, strategic alignment

## Communication Plan

- Daily: task updates and blockers
- Weekly: progress, risks, workload, decisions
- Sprint review: completed work, metrics, next priorities
